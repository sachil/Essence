# Essence

**Essence**是一款使用[干货集中营](https://gank.io)提供的API所开发的APP，它使用**Kotlin**语言开发，网络请求使用**协程+retrofit**，代码架构使用当前最流行的**MVVM**架构。本项目采用开源的方式，目的是为了帮助更多的初学者来了解`Kotlin`语言以及`MVVM`架构。
